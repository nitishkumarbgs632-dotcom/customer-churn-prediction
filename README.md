# Customer Churn Prediction System

## Overview

This is an end-to-end machine learning project that predicts whether a customer is likely to churn. The system uses a trained model and provides insights to help businesses improve customer retention.

## My Contribution (Team Project)

This project was developed as part of a team. My contributions include:

* Data preprocessing and cleaning
* Understanding and working with the machine learning model
* Testing the prediction output and analyzing results

## Features

* Predicts customer churn probability
* Uses Random Forest classification model
* Provides basic insights based on prediction

## Technologies Used

* Python
* Machine Learning
* Pandas, Scikit-learn
* FastAPI (basic understanding)
* React (basic understanding)

## Learning Outcome

Through this project, I learned how machine learning models are used in real-world applications and how frontend, backend, and ML systems are connected.
